
# Novation-Ultranova Sample Library

A digita synth similar to the older Nova & SuperNova devices. 

-  **Patches**: 20
-   **Formats:** EXS24, Kontakt, NN-XT & SFZ
-   **Approx Size:** 6GB
-   **Info:** [novationmusic.com](https://novationmusic.com/en/synths/)
